# Startup manual

### Installation

Clone the repo and install dependencies:

```bash
cd ./backend
uv sync

cd ../fronted
pnpm i
```

### Start project

```bash
sh ./start.sh
```

# Description

#### Fullstack app for load posts from inst and filter them.
